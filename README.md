# Space UI [![Build Status](https://travis-ci.org/CodeAdventure/meteor-space-ui.svg?branch=master)](https://travis-ci.org/CodeAdventure/meteor-space-ui)

Reactive UI based on Iron Router and the Flux architecture.

## Install
`meteor add space:ui`

## Run the tests
`mrt test-packages ./`
