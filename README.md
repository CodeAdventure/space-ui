## Documentation
API Reference and User Guide are coming soon

## Installation
```
meteor add space:ui
meteor add peerlibrary:blaze-components
```
_Implementing [flux](https://meteor-space.readme.io/docs/flux-in-depth)?_
```
meteor add space:flux
```
